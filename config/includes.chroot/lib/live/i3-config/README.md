i3 config
=========
All file config for i3 window manager.

I use Debian GNU/Linux.
```
$i3 --version
i3 version 4.13 (2016-11-08) © 2009 Michael Stapelberg and contributors
```

## how to setup
```
git clone https://github.com/TxGVNN/i3-config.git ~/.i3
```
